    3. Usage – Customization

    3.1 HTML

    Now comes the really interesting part! Learn how to use and customize this website template to fit your needs. The most important thing for you to do is actually look into the HTML source code and see how elements are written! The clean and commented code makes it easy for you to copy code blocks from content modules and past them in different places or layouts.

    <!-- ============================================== LOOKBOOK SLIDER ============================================== -->
<div class="lookbook-slider">
	...
</div>
<!-- ============================================== LOOKBOOK SLIDER : END ============================================== -->

    3.2 Logo

    It is highly recommended to use vector graphics for the logo like the Bewear template does with the PNG file format. This ensures the best possible and sharpest view on mobile devices and of course when scaling the logo. If it is not possible for you to use vector graphics you can still use raster graphics like the JPG or PNG file format of course. In this case you have to change the filename extension of the logo in the following HTML parts to this for example:

    

    
<!-- ========================================= LOGO ========================================= -->

<div class="container ">	
	<a href="index.php?page=home" class="navbar-brand">
		<img alt="" src="assets/images/logo.png" class="logo" />
	</a>	
</div>
<!-- ========================================= LOGO : END ========================================= -->

    3.2 Images

    The image usage in this website template is very easy to handle as well. The various image dimensions used across the website has been given as a placholder. The placeholder will also contain the image dimensions. This service is provided by http://placehold.it. Here is an example below :

    <div class="featured-product text-center">
	<div class="image lazy-load">
		<a href="index.php?page=product-simple">
		    <img class="img-responsive" alt="" src="http://placehold.it/300x325/" />
		</a>
	</div>
</div>
